# 学生成绩管理系统 python82

## 项目概述

学生成绩管理系统是一款基于Python语言，使用PyQt5作为图形用户界面，MySQL作为数据库管理系统的软件。该系统主要用于高校或教育机构管理学生成绩和相关信息。

## 技术栈

- 开发语言：Python 3.7
- 开发工具：PyCharm、Anaconda
- 图形界面库：PyQt5
- 数据库：MySQL

## 功能模块

### 系统角色

- 管理员

#### 功能介绍

1. 管理员登录与信息修改
2. 学生管理：增删改查，支持批量导入导出
3. 成绩管理：成绩录入、查询、修改、统计，支持批量导入导出
4. 班级管理：增删改查，支持批量导入导出
5. 角色管理（教师管理）：教师信息管理，班级权限赋予

### 数据库结构

- 管理员：用户名、密码、备注、可管理班级ID
- 学生信息：姓名、学号、性别、所属班级ID、语文成绩、数学成绩、外语成绩
- 班级信息：班级名称

## 运行环境

- Python 3.7
- PyQt5
- MySQL

## 部署步骤

1. 确保Python 3.7环境以及PyQt5、MySQL已正确安装
2. 导入数据库结构和数据
3. 运行主程序

```
.

```
